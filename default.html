HTML

The language that the Web is written in.

by Charles Iliya Krempeaux

There are a small set of core technologies that is the Web is made out of — HTML is one of them.

In fact, in the beginning there was just HTML and HTTP. Later, came (non-animated) GIF raster images (in January 1993), Cookies (in October 1994), JavaScript (originally named LiveScript) (in September 1995), animated GIF raster images (in December 1995), Bookmarklets (in March 1996), Flash (originally named FutureSplash) (in May 1996), CSS (in August 1996), JPEG raster images (in August 1996), PNG raster images (in September 1997), Web Fonts (in September 1997), XML (in March 1999) XMLHttpRequest (in March 1999) XSLT (in November 2000), SVG vector images (in February 2004), RSS web feeds (in September 2004), Atom web feeds (in April 2005), <video> (in June 2009), Ogg Theora videos (in June 2009), <audio> (in June 2009), MP3 audios (in June 2009), AAC audios (in June 2009), Ogg Vorbis audios (in June 2009), WebSockets (in December 2009), MP4 (H.264) videos (in April 2010), WebM videos (in May 2010), WebP raster images (in September 2010), WebRTC (in January 2012), WebAssembly (in March 2017), AVIF raster images (in August 2020), WebGPU (in May 2023), etc.

But, this article is about HTML, so let's focus on that.

Paired HTML Tags

HTML looks like this:

It looks like regular written language, except for the <b> and </b>. Those are called HTML tags. <b> is an example of an opening HTML tag. And, </b> is an example of an closing HTML tag. Opening and closing HTML tags look very similar except that the closing HTML tag has a (forward) slash (i.e., a solidus) as the second character.

Here are some more opening and closing HTML tags:

  • <abbr> </abbr>
  • <b> </b>
  • <cite> </cite>
  • <code> </code>
  • <date> </data>
  • <dfn> </dfn>
  • <em> </em>
  • <i> </i>
  • <kbd> </kbd>
  • <mark> </mark>
  • <s> </s>
  • <samp> </samp>
  • <small> </small>
  • <span> </span>
  • <strong> </strong>
  • <sub> </sub>
  • <sup> </sup>
  • <time> </time>
  • <u> </u>
  • <var> </var>

There are a lot more HTML tags than just these.

The opening and closing HTML tags work together to markup the text. In our previous example, the <b> and </b> worked together to markup the text: “you”. What the <b> and </b> HTML tags do is make text bold.

Unpaired HTML Tags

The HTML tags we just looked at all came in pairs — with an opening HTML tag (such as <b>, <i>, and <sub>) and a closing HTML tag (such as </b>, </i>, and </sub>).

And even though most HTML tags are paired, some HTML tags are not paired.

(Note that you will sometimes hear or see these unpaired HTML tags called: void elements. In this case, the “void” in “void element” meaning that there is nothing inside of them.)
