HTML

The language that the Web is written in.

by Charles Iliya Krempeaux

There are a small set of core technologies that is the Web is made out of — HTML is one of them.

In fact, in the beginning (with respect to the core technologies of the Web) there was just HTML and HTTP. Later, came (non-animated) GIF raster images (in January 1993), Cookies (in October 1994), JavaScript (originally named LiveScript) (in September 1995), animated GIF raster images (in December 1995), Flash (originally named FutureSplash) (in May 1996), CSS (in August 1996), JPEG raster images (in August 1996), PNG raster images (in September 1997), Web Fonts (in September 1997), XMLHttpRequest (in March 1999) SVG vector images (in February 2004), <video> (in June 2009), Ogg Theora videos (in June 2009), <audio> (in June 2009), MP3 audios (in June 2009), AAC audios (in June 2009), Ogg Vorbis audios (in June 2009), WebSockets (in December 2009), MP4 (H.264) videos (in April 2010), WebM videos (in May 2010), WebP raster images (in September 2010), WebRTC (in January 2012), WebAssembly (in March 2017), AVIF raster images (in August 2020), WebGPU (in May 2023), etc. But, this article is about HTML, so let's focus on that.
